# Interactive Mascot-Based Developer Portfolio

A professional yet playful portfolio website featuring an interactive fluffy mascot that guides users through skills, projects, and personality using a deeply immersive Audio-Visual (AV) system.

## Features

- Fluffy Baymax-inspired mascot with eye tracking and emotional states
- Sound-reactive particle system
- Custom cursor with trailing effects
- GSAP-powered animations
- Supabase backend integration
- GitHub API integration for projects
- Accessibility-first design

## Tech Stack

- **Build**: Vite + TypeScript
- **3D/Animation**: Three.js, GSAP, Framer Motion, Anime.js
- **Audio**: Howler.js, Web Audio API
- **Styling**: TailwindCSS
- **State**: Zustand
- **Backend**: Supabase

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` to `.env` and fill in your credentials.
